Project Details

This project was created for a 36 hour continous coding competition (hackathon). Our problem statement was: Government Subsidy Blockchain. We had to create an e-ledger based blockchain that would ensure transparency in the domain of Goverment Subsidies. This was to be implemented in the form of a website or an application. We built a website hosted on our localhost server for demonstration purposes. Every transaction logged through the portal would be hashed and appended to a blockchain. If there was any corruption or wrongdoing it would easily be caught as the hash value would change. We placed in the top 10 in this competition, out of around 100 teams.
