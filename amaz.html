Project Details

I lead a team of 2 to create a clone of the popular retail website: Amazon. Our aim was to create a website that looked and functioned almost exactly like the real one did. We used HTML, CSS, JS and PHP to create this particular project. All sign-ins by customers were added to a MySQL database and users remained logged in as we used PHP sessions to ensure just that. This project was part of my Internet Programming course, and was hosted shortly at clone.amaze.in. My external examiner was so impressed with this replica, she said she was afraid that we would get a copyright notice from amazon for using their logo and UI.
